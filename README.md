# Weddingapp2 Local Recreation

This is a standalone static recreation inspired by `https://weddingapp2.pages.dev/`.

Open `index.html` in a browser to view it. The page includes:

- cinematic blue sky intro
- shooting stars and couple reveal
- Explore transition
- scrollable invitation page
- scratch-to-reveal card
- countdown
- location illustration
- happy memories gallery
- tap/click butterfly effects

Edit names, countdown date, and map URL at the top of `script.js`.
